Comandos Disponíveis

        
    
          
  • moverEsquerda() — Move o robô 1 casa para a esquerda
    • 
          
  • moverDireita() — Move o robô 1 casa para a direita
    • 
          
  • moverCima() — Move o robô 1 casa para cima
    • 
          
  • moverBaixo() — Move o robô 1 casa para baixo
    • 
        


        

          O robô também pode se mover com parâmetros. Exemplo: (x) = quantidade de casas.

          Para avançar apenas uma casa, deixe o parâmetro vazio: ()

          Escreva um comando por linha no editor.

          O robô reconhece apenas os quatro comandos disponíveis no tutorial.
